== Description ==

Hides YouTube Related Videos!  It is basically like adding the "?rel=0" to the end of youtube embedded links.  This very lightweight wordpress plugin turns off the annoying videos that pop up when a user pauses or finishes a embedded YouTube video.  You just activate YTR0 and all of your embedded YouTube videos will look awesome.
